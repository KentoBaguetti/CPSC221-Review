# Graph Notes for 221
### What is a Graph?
Collection of vertices (nodes) connected via edges. <br>
**G = (V, E)** <br>
**V** is a set of vertices; {v_1, v_2, v_3, ..., v_n} <br>
**E** is a set of edges; {e_1, e_2, e_3, ..., e_m} where each e_i is a pair of vertices: e_i = (V x V) <br>
